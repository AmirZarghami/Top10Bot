# Top10Bot
The First Version of the Top10Bot, Finally Released

A terminal-based Python program to manage a personal “Top 10 songs” project, store profiles, and generate stats/awards.

## Requirements
- Python **3.10+**
- Works on Windows / macOS / Linux (terminal required)
